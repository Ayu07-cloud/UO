First: Understand Polar Form & Euler's Formula

The FOUNDATION — everything else builds on this
🧒 Think of it this way

A complex number a + ib is like a point on a map. Instead of saying "3 steps right, 4 steps up", you can say "5 steps in a certain direction". The 5 is called r (modulus) and the direction is called θ (argument). That's all polar form is — a different GPS coordinate for the same point!

📖 The Formal Definition

Every complex number z = a + ib can be written as z = r(cosθ + i sinθ) where r = |z| = √(a²+b²) is the modulus and θ = arg(z) = tan⁻¹(b/a) is the argument. Euler's formula then gives us the compact form z = re^(iθ).

🌟 Euler's Formula — The Most Important Formula in All of Maths
e^(iθ) = cos θ + i sin θ Special cases (memorize these!): e^(iπ) = cos π + i sin π = -1 + 0 = -1 e^(iπ/2) = cos(π/2) + i sin(π/2) = 0 + i = i e^(i·0) = cos 0 + i sin 0 = 1 + 0 = 1 General polar form: a + ib = r·e^(iθ) where r = √(a²+b²), θ = tan⁻¹(b/a)
🧒 Why does e^(iθ) = cosθ + i sinθ? (Toddler Proof)

Expand e^(iθ) using the Taylor series: e^x = 1 + x + x²/2! + x³/3! + ...
Put x = iθ: e^(iθ) = 1 + iθ + (iθ)²/2! + (iθ)³/3! + ...
Now i² = -1, i³ = -i, i⁴ = 1 — these cycle! Grouping real and imaginary parts gives exactly cosθ + i sinθ. Mind = blown. 🤯

De Moivre's Theorem

THE most important theorem in Unit I — appears EVERY exam
🧒 The Spinning Clock Analogy

Imagine a clock hand pointing at angle θ. If you spin it to that position n times in a row, where does it end up? At angle n×θ — obviously! De Moivre says the same thing for complex numbers. Multiplying (cosθ + i sinθ) by itself n times just multiplies the angle by n. That's literally all it is.

📖 Formal Statement

For any rational number n (integer, fraction), De Moivre's Theorem states:

(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)

Using Euler: Since e^(iθ) = cosθ + i sinθ, raising to power n gives e^(inθ) = cos(nθ) + i sin(nθ). The theorem is literally just a restatement of Euler + exponent rules!

⚡ De Moivre's Theorem — The Formula
(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ) Also written as: (e^iθ)ⁿ = e^(inθ) Works for ALL rational n: integers, fractions, negative numbers!
PROOF — BY INDUCTION
Base Case (n=2): (cosθ + i sinθ)² = cos²θ + 2i sinθ cosθ + i²sin²θ = (cos²θ - sin²θ) + i(2sinθ cosθ) = cos(2θ) + i sin(2θ) ✓ [using double angle formulas] Inductive Step: Assume true for n=k: (cosθ + i sinθ)^k = cos(kθ) + i sin(kθ) For n = k+1: (cosθ + i sinθ)^(k+1) = [cos(kθ) + i sin(kθ)] · [cosθ + i sinθ] = cos(kθ)cosθ - sin(kθ)sinθ + i[sin(kθ)cosθ + cos(kθ)sinθ] = cos(kθ + θ) + i sin(kθ + θ) [using addition formulas] = cos(k+1)θ + i sin(k+1)θ ✓ For negative n: Use (cosθ + i sinθ)⁻¹ = cos(-θ) + i sin(-θ)
🎯 PATTERN RECOGNITION — How to identify a De Moivre question
IF YOU SEE
"Prove that cos 5θ = 16cos⁵θ - 20cos³θ + 5cosθ" → De Moivre to expand (cosθ+i sinθ)⁵, then take real part
IF YOU SEE
"Find all nth roots of a complex number" → Write in polar form, use De Moivre with power 1/n, substitute k = 0,1,...,n-1
IF YOU SEE
"Express cosⁿθ or sinⁿθ in multiple angles" → Use De Moivre: write (cosθ+i sinθ)ⁿ, expand by binomial theorem, compare parts
IF YOU SEE
"Show that..." with trig involving nθ → De Moivre is the tool — convert to exponential, simplify, convert back
📋 PYQ Questions — De Moivre's Theorem
Q1
Prove De Moivre's Theorem and use it to expand cos(5θ) in powers of cos θ
🔥 MUST DO Asked 2019, 2021, 2023 14 Marks
▾
🧒 What this question is really asking

Part 1: Prove the theorem (we did that above). Part 2: Find cos(5θ) as a polynomial in cosθ only. The trick is to expand (cosθ+i sinθ)⁵ using Binomial Theorem, then compare REAL parts on both sides.

✏️ Step-by-Step Solution
  • 1
    Write the De Moivre form
    (cosθ + i sinθ)⁵ = cos(5θ) + i sin(5θ) ...(by De Moivre)
  • 2
    Expand the LEFT side using Binomial Theorem (a+b)⁵ Let a = cosθ, b = i sinθ. Use ⁵C₀, ⁵C₁... = 1, 5, 10, 10, 5, 1
    = ⁵C₀cos⁵θ + ⁵C₁cos⁴θ(isinθ) + ⁵C₂cos³θ(isinθ)² + ⁵C₃cos²θ(isinθ)³ + ⁵C₄cosθ(isinθ)⁴ + ⁵C₅(isinθ)⁵ Remember: i²=-1, i³=-i, i⁴=1, i⁵=i = cos⁵θ + 5i cos⁴θ sinθ - 10cos³θ sin²θ - 10i cos²θ sin³θ + 5cosθ sin⁴θ + i sin⁵θ
  • 3
    Group REAL parts (no i) and IMAGINARY parts (with i)
    Real part = cos⁵θ - 10cos³θ sin²θ + 5cosθ sin⁴θ Imag part = 5cos⁴θ sinθ - 10cos²θ sin³θ + sin⁵θ
  • 4
    Compare Real parts from both sides. Then substitute sin²θ = 1-cos²θ to get ALL cosines.
    cos(5θ) = cos⁵θ - 10cos³θ(1-cos²θ) + 5cosθ(1-cos²θ)² Expand: = cos⁵θ - 10cos³θ + 10cos⁵θ + 5cosθ(1 - 2cos²θ + cos⁴θ) = cos⁵θ - 10cos³θ + 10cos⁵θ + 5cosθ - 10cos³θ + 5cos⁵θ = 16cos⁵θ - 20cos³θ + 5cosθ
✅ FINAL ANSWER

cos(5θ) = 16cos⁵θ − 20cos³θ + 5cosθ
Similarly, comparing imaginary parts gives: sin(5θ) = 16sin⁵θ - 12sin³θ + ... (can be asked separately)

💡 THE GOLDEN RULE for these questions

Real part comparison → cos(nθ) formula
Imaginary part comparison → sin(nθ) formula
After grouping, ALWAYS replace sin²θ with (1−cos²θ) to get cos only (or vice versa). The binomial coefficients ⁵C₀=1, ⁵C₁=5, ⁵C₂=10, ⁵C₃=10, ⁵C₄=5, ⁵C₅=1 — memorize Pascal's triangle!

nth Roots of Complex Numbers & Roots of Unity

Cube roots, 4th roots — asked almost every single year
🧒 The Pizza Slice Analogy

Finding the nth roots of a complex number is like cutting a circular pizza into n equal slices. Each slice's starting corner is one root! They're all equally spaced around a circle at angles 360°/n apart. Cube roots → 3 slices (0°, 120°, 240°). 4th roots → 4 slices (0°, 90°, 180°, 270°). Simple!

📖 The Method (3 steps)

Step 1: Write the complex number in polar form: r·e^(iθ)
Step 2: The general form adds 2kπ to angle: r·e^(i(θ+2kπ))
Step 3: Apply power 1/n and substitute k = 0, 1, 2, ..., n−1

You always get exactly n distinct roots.

🔑 Master Formula for nth Roots
If z = r(cosθ + i sinθ), then the n roots of z^(1/n) are: z_k = r^(1/n) · [cos((θ + 2kπ)/n) + i sin((θ + 2kπ)/n)] where k = 0, 1, 2, ..., (n-1) For roots of UNITY (z^n = 1): r=1, θ=0 z_k = cos(2kπ/n) + i sin(2kπ/n) = e^(2kπi/n)
🎯 HOW TO INSTANTLY IDENTIFY THESE QUESTIONS
KEYWORDS
"roots of unity", "find all values of z where zⁿ = 1", "cube root of -1 or i", "nth roots of a+ib" → Always use the master formula above with k=0,1,...,n-1
TRICK
For cube roots of unity specifically, the 3 roots are always: 1, ω, ω² where ω = e^(2πi/3). The magic property 1+ω+ω²=0 is asked in EVERY CT!
TRICK
For 4th roots of 1: roots are 1, i, -1, -i. For 4th roots of -1: write -1 = e^(iπ), then use k=0,1,2,3.
📋 PYQ Questions — nth Roots
Q2
Find all the cube roots of unity. Show that their sum is 0 and product is 1.
🔥 MUST DO Asked Every Year (CT + ESE) 7–14 Marks
▾
🧒 Setup — what are we solving?

We want z such that z³ = 1. A cubic equation has exactly 3 roots. We need to find all 3 using the master formula. The number 1 in polar form is r=1, θ=0.

✏️ Step-by-Step Solution
  • 1
    Write 1 in general polar form (add 2kπ)
    1 = cos(0) + i sin(0) = cos(2kπ) + i sin(2kπ) for k = 0, 1, 2, ...
  • 2
    Apply De Moivre with power 1/3
    z = (1)^(1/3) = [cos(2kπ) + i sin(2kπ)]^(1/3) = cos(2kπ/3) + i sin(2kπ/3)
  • 3
    Substitute k = 0, 1, 2 (only 3 distinct roots)
    k=0: z₁ = cos(0) + i sin(0) = 1 k=1: z₂ = cos(2π/3) + i sin(2π/3) = -1/2 + i(√3/2) ← this is called ω k=2: z₃ = cos(4π/3) + i sin(4π/3) = -1/2 - i(√3/2) ← this is ω²
  • 4
    Prove Sum = 0
    1 + ω + ω² = 1 + (-1/2 + i√3/2) + (-1/2 - i√3/2) = 1 - 1/2 - 1/2 + i(√3/2 - √3/2) = 0 + 0i = 0 ✓
  • 5
    Prove Product = 1
    1 · ω · ω² = ω³ = (e^(2πi/3))³ = e^(2πi) = cos(2π) + i sin(2π) = 1 ✓
✅ RESULTS TO MEMORIZE

Three cube roots of unity: 1, ω = (-1+i√3)/2, ω² = (-1-i√3)/2
Sum: 1 + ω + ω² = 0 (always!)
Product: 1 · ω · ω² = 1
Key: ω³ = 1 and ω is a root of z²+z+1=0

Q3
Find all fourth roots of (-1) and represent them on an Argand diagram.
🟡 High Priority Asked 2020, 2022 7 Marks
▾
✏️ Step-by-Step Solution
  • 1
    Write -1 in polar form
    -1 = cos(π) + i sin(π) = cos(π + 2kπ) + i sin(π + 2kπ) (r = 1, θ = π, and we add 2kπ for general form)
  • 2
    Apply De Moivre with power 1/4
    z = (-1)^(1/4) = cos((π + 2kπ)/4) + i sin((π + 2kπ)/4)
  • 3
    Substitute k = 0, 1, 2, 3
    k=0: z₁ = cos(π/4) + i sin(π/4) = 1/√2 + i/√2 k=1: z₂ = cos(3π/4) + i sin(3π/4) = -1/√2 + i/√2 k=2: z₃ = cos(5π/4) + i sin(5π/4) = -1/√2 - i/√2 k=3: z₄ = cos(7π/4) + i sin(7π/4) = 1/√2 - i/√2
  • 4
    Argand Diagram description 4 points equally spaced at 90° intervals on a circle of radius 1, starting at 45° (π/4). Like a compass with NE, NW, SW, SE directions.
✅ FOUR ROOTS

z = ±(1±i)/√2 — all four combinations of ± signs

💡 PATTERN TRICK

For the 4th roots of 1: they are 1, i, -1, -i (the axes points).
For 4th roots of -1: rotate all by 45° → you get the diagonal points above.
General rule: nth roots of 1 are equally spaced at 2π/n, starting at 0°. Roots of -1 start at π/n.

Separating Real & Imaginary Parts

log(a+ib), sin(a+ib), cos(a+ib), sinh, cosh — all use the same 2 tricks
🧒 The Recipe Book Analogy

When you write log(a+ib) or sin(a+ib), you have a complex argument inside a function. To separate them, you just need 2 tools:
Tool 1: Write a+ib in polar form as r·e^(iθ)
Tool 2: Know the "expansion formulas" for sin/cos/log of complex numbers
After applying the formula, group terms with 'i' (imaginary) and without 'i' (real). Done!

🧮 Master Formulas — Memorize ALL of these

log of complex
log(a+ib) = ½log(a²+b²) + i·tan⁻¹(b/a) Real = ½log(a²+b²) Imag = tan⁻¹(b/a)
sin of complex
sin(α+iβ) = sinα·coshβ + i·cosα·sinhβ Real = sinα·coshβ Imag = cosα·sinhβ
cos of complex
cos(α+iβ) = cosα·coshβ - i·sinα·sinhβ Real = cosα·coshβ Imag = -sinα·sinhβ
hyperbolic defs
sinh(x) = (eˣ - e⁻ˣ)/2 cosh(x) = (eˣ + e⁻ˣ)/2 sin(iθ) = i·sinh(θ) cos(iθ) = cosh(θ)
tan of complex
tan(α+iβ) = [sin2α + i·sinh2β] / [cos2α + cosh2β]
exponential
e^(a+ib) = eᵃ(cosb + i sinb) Real = eᵃ·cosb Imag = eᵃ·sinb
🎯 HOW TO RECOGNISE & SOLVE INSTANTLY
SEE log(a+ib)
DIRECTLY use: Real = ½log(a²+b²), Imag = tan⁻¹(b/a). No steps needed, just plug in!
SEE sin(A+iB) or cos(A+iB)
Use the expansion formulas above. Identify α and β, plug in to the formula.
SEE (a+ib)^(c+id)
Write as e^((c+id)·log(a+ib)), then expand. First find log(a+ib), multiply by (c+id), then use e^(x+iy) formula.
📋 PYQ Questions — Real & Imaginary Parts
Q4
Separate log(3 + 4i) into real and imaginary parts. Also separate (1+i)^(1+i).
🟡 High Priority Asked 2021, 2022 7 Marks
▾
✏️ Part A: log(3+4i)
  • 1
    Apply the log formula directly
    log(a+ib) = ½log(a²+b²) + i·tan⁻¹(b/a) Here a=3, b=4: log(3+4i) = ½log(3²+4²) + i·tan⁻¹(4/3) = ½log(25) + i·tan⁻¹(4/3) = ½log(25) + i·tan⁻¹(4/3)
✅ PART A ANSWER

Real part = ½ log 25 = log 5
Imaginary part = tan⁻¹(4/3)

✏️ Part B: (1+i)^(1+i) — the trickier one
  • 1
    Write as exponential: (1+i)^(1+i) = e^((1+i)·log(1+i))
  • 2
    Find log(1+i): here a=1, b=1
    log(1+i) = ½log(1²+1²) + i·tan⁻¹(1/1) = ½log(2) + i·(π/4)
  • 3
    Multiply (1+i) × log(1+i)
    (1+i)(½log2 + iπ/4) = ½log2 + iπ/4 + i·½log2 + i²π/4 = ½log2 - π/4 + i(π/4 + ½log2)
  • 4
    Now use e^(x+iy) = eˣ(cosy + i siny)
    x = ½log2 - π/4, y = π/4 + ½log2 (1+i)^(1+i) = e^(½log2 - π/4) · [cos(π/4 + ½log2) + i sin(π/4 + ½log2)]
✅ PART B ANSWER

Real = e^(½log2 - π/4) · cos(π/4 + ½log2)
Imaginary = e^(½log2 - π/4) · sin(π/4 + ½log2)

C+iS Method for Summing Series

THE smartest trick in Complex Numbers — always asked in ESE
🧒 Why does C+iS work?

Suppose you're asked to sum: cosθ + cos2θ + cos3θ + ...
Alone, this is hard. BUT if you create a "sister series" with sin, together they form e^(iθ) + e^(2iθ) + ... which is just a geometric series! Easy to sum! After summing, take real part → you get the cosine sum. Take imaginary part → you get the sine sum. Two birds, one geometric series!

📖 The Method — 5 Fixed Steps

Step 1: Define C = cos series, S = sin series
Step 2: Form C + iS (a geometric series in e^(iθ))
Step 3: Sum using GP formula: S = a(1−rⁿ)/(1−r)
Step 4: Rationalize denominator (multiply by conjugate)
Step 5: Separate real and imaginary parts

🔑 Key: Geometric Series Formula
Sum of GP: a + ar + ar² + ... + arⁿ⁻¹ = a(1 - rⁿ)/(1 - r) In C+iS: a = e^(iθ), r = e^(iθ) C + iS = e^(iθ)[1 - e^(inθ)] / [1 - e^(iθ)] KEY TRICK to simplify: e^(iA) - e^(iB) = e^(i(A+B)/2) · 2i·sin((A-B)/2) 1 - e^(iA) = -e^(iA/2) · 2i·sin(A/2)
📋 PYQ Questions — C+iS Method
Q5
Find the sum: C = cosθ + cos2θ + cos3θ + ... + cosnθ using C+iS method
🔥 MUST DO Asked 2019, 2020, 2023 14 Marks
▾
✏️ Full Solution
  • 1
    Define C and S, form C+iS
    C = cosθ + cos2θ + ... + cosnθ S = sinθ + sin2θ + ... + sinnθ C + iS = (cosθ+isinθ) + (cos2θ+isin2θ) + ... + (cosnθ+isinnθ) = e^(iθ) + e^(2iθ) + ... + e^(niθ)
  • 2
    This is a GP: first term a = e^(iθ), ratio r = e^(iθ), n terms
    C + iS = e^(iθ) · [1 - e^(niθ)] / [1 - e^(iθ)]
  • 3
    Factor e^(inθ/2) from numerator, e^(iθ/2) from denominator
    Numerator: 1 - e^(niθ) = -e^(inθ/2)[e^(inθ/2) - e^(-inθ/2)] = -e^(inθ/2) · 2i·sin(nθ/2) Denominator: 1 - e^(iθ) = -e^(iθ/2) · 2i·sin(θ/2) C + iS = e^(iθ) · [-e^(inθ/2)·2i·sin(nθ/2)] / [-e^(iθ/2)·2i·sin(θ/2)] = e^(iθ) · e^(inθ/2) · sin(nθ/2) / [e^(iθ/2) · sin(θ/2)] = e^(iθ/2 + inθ/2) · sin(nθ/2) / sin(θ/2) = e^(i(n+1)θ/2) · sin(nθ/2) / sin(θ/2)
  • 4
    Expand e^(i(n+1)θ/2) = cos((n+1)θ/2) + i sin((n+1)θ/2) and separate
    C + iS = [sin(nθ/2)/sin(θ/2)] · [cos((n+1)θ/2) + i sin((n+1)θ/2)] Comparing REAL parts: C = [sin(nθ/2) / sin(θ/2)] · cos[(n+1)θ/2] Comparing IMAGINARY parts: S = [sin(nθ/2) / sin(θ/2)] · sin[(n+1)θ/2]
✅ FINAL ANSWERS

cosθ + cos2θ + ... + cosnθ = [sin(nθ/2) / sin(θ/2)] · cos[(n+1)θ/2]
sinθ + sin2θ + ... + sinnθ = [sin(nθ/2) / sin(θ/2)] · sin[(n+1)θ/2]

💡 EXAM SHORTCUT FOR C+iS

The factoring step (Step 3) is the hardest. Remember this pattern:
1 − e^(iA) = −e^(iA/2) × 2i × sin(A/2)
This one identity is the key to ALL C+iS problems. Write it on your rough sheet first!

What is a Series? What is Convergence?

The foundation — don't skip this even if you're in a hurry
🧒 Infinite Sum Analogy

Imagine walking toward a wall, but each step is half the previous one: 1m, ½m, ¼m, ⅛m... You keep walking forever, but you never go past 2m! That's a convergent series — adding infinitely many terms gives a finite answer. But if each step is 1m, 1m, 1m... forever → you never stop → that's divergent. The test is: does the sum settle down or blow up?

📖 Formal Definition

A series Σuₙ = u₁ + u₂ + u₃ + ... is called convergent if the partial sums Sₙ = u₁+u₂+...+uₙ tend to a finite limit as n→∞. It's divergent if Sₙ→∞ or oscillates. A necessary condition for convergence: uₙ→0 as n→∞ (but NOT sufficient — the harmonic series 1+½+⅓+¼+... diverges even though uₙ→0!)

⚠️ COMMON EXAM TRAP
If uₙ → 0, that does NOT guarantee convergence! The harmonic series Σ(1/n) = 1 + 1/2 + 1/3 + ... DIVERGES even though 1/n → 0. Always use a proper test — never just check if uₙ→0.
🗺️ WHICH TEST TO USE — Decision Flowchart
You have a series Σuₙ. Which test to use?
↓
Does the series contain: factorial (n!), exponential (aⁿ), or products like n·(n-1)·...?
↓
YES → Try Ratio Test first
↓
Ratio Test gives L=1 (fails)?
↓
Use Raabe's Test instead
Contains nth power like (2n/(3n+1))ⁿ
↓
Use Cauchy's Root Test
↓
Simple algebraic terms like n²/(n³+1)?
↓
Comparison Test or Limit Comparison with known series (like 1/n, 1/n²)

D'Alembert's Ratio Test

The KING of convergence tests — appears 100% of the time in CSVTU exams
🧒 The Shrinking Steps Analogy

You're adding steps to a staircase. The Ratio Test asks: "Is each new step shorter than the last?" If each step is less than some fraction L of the previous one (L < 1), you'll never reach infinity — convergent! If each step is bigger (L > 1), you climb forever — divergent! If exactly equal (L=1), we can't tell — need another test.

📖 The Statement

For series Σuₙ of positive terms, compute:
L = lim(n→∞) |u(n+1)/uₙ|

• If L < 1 → Series CONVERGES
• If L > 1 → Series DIVERGES
• If L = 1 → Test FAILS, try Raabe's

Works best when uₙ has factorials, exponentials, or products.

🎯 PATTERN RECOGNITION — Ratio Test Questions
CLUE 1
Series contains n! (factorial) → Definitely Ratio Test. Factorial in ratio simplifies beautifully: (n+1)!/n! = (n+1)
CLUE 2
Series has terms like xⁿ, 2ⁿ, 3ⁿ → Ratio Test. The exponential cancels neatly.
CLUE 3
Series of form Σ n·xⁿ/aⁿ → Ratio Test to find radius of convergence
WATCH OUT
Series like Σ [1·3·5...(2n-1)] / [2·4·6...(2n)] → Ratio Test gives L=1 → MUST use Raabe's!
📋 PYQ Questions — Ratio Test
Q6
Test the convergence of: Σ n!/nⁿ
🔥 MUST DO Classic CSVTU — every year 7 Marks
▾
🧒 Why is this a Ratio Test question?

It has n! (factorial) — that's the #1 signal for Ratio Test. The factorial in numerator/denominator will simplify perfectly when you form uₙ₊₁/uₙ.

✏️ Step-by-Step
  • 1
    Write uₙ and uₙ₊₁
    uₙ = n!/nⁿ uₙ₊₁ = (n+1)!/(n+1)^(n+1)
  • 2
    Form uₙ₊₁/uₙ (this is the ONLY calculation)
    uₙ₊₁/uₙ = [(n+1)!/(n+1)^(n+1)] × [nⁿ/n!] KEY STEP: (n+1)!/n! = (n+1) ← factorial simplification! = (n+1) × nⁿ / (n+1)^(n+1) = nⁿ / (n+1)ⁿ = [n/(n+1)]ⁿ = [1 / (1 + 1/n)]ⁿ
  • 3
    Take the limit (use the standard limit result)
    L = lim(n→∞) [1/(1+1/n)]ⁿ STANDARD RESULT: lim(n→∞) (1 + 1/n)ⁿ = e Therefore: lim(n→∞) [1/(1+1/n)]ⁿ = 1/e ≈ 0.368 L = 1/e < 1
  • 4
    State the conclusion Since L = 1/e < 1, by D'Alembert's Ratio Test, the series CONVERGES.
✅ ANSWER

Σ n!/nⁿ CONVERGES (L = 1/e < 1 by Ratio Test)

💡 THE KEY RESULT TO MEMORIZE

lim(n→∞) (1 + 1/n)ⁿ = e — This appears in ALMOST EVERY ratio test problem involving nⁿ. Recognizing it saves you 3 minutes in the exam!

Q7
Test convergence of: Σ (n² · 2ⁿ) / n!
🟡 High Priority 2022, 2023 7 Marks
▾
✏️ Step-by-Step
  • 1
    Write uₙ and uₙ₊₁
    uₙ = n²·2ⁿ/n! uₙ₊₁ = (n+1)²·2^(n+1)/(n+1)!
  • 2
    Form the ratio and simplify
    uₙ₊₁/uₙ = [(n+1)²·2^(n+1)/(n+1)!] × [n!/(n²·2ⁿ)] = [(n+1)²/n²] × [2^(n+1)/2ⁿ] × [n!/(n+1)!] = [(n+1)²/n²] × 2 × [1/(n+1)] = 2(n+1)/n²
  • 3
    Take limit
    L = lim(n→∞) 2(n+1)/n² = lim(n→∞) 2/n = 0 L = 0 < 1 → CONVERGES
✅ ANSWER

Series CONVERGES (L = 0 < 1)

🔬

Raabe's Test — The Backup to Ratio Test

Use ONLY when D'Alembert gives L=1. Often asked with product-type series.
🧒 Why do we need Raabe's Test?

When the Ratio Test gives L=1, it's like a judge saying "I can't decide!" Raabe's test is the higher court — it looks MORE carefully at how the ratio approaches 1. Specifically, it checks whether n × (1 - ratio) converges to something greater or less than 1. If it's >1, the series converges. If <1, it diverges.

🔬 Raabe's Test — When to use & How
USE ONLY WHEN Ratio Test gives L = 1. Compute: R = lim(n→∞) n · [1 - u(n+1)/uₙ] • If R > 1 → Series CONVERGES • If R < 1 → Series DIVERGES • If R = 1 → Test FAILS (use logarithmic test — rare in exams)
🎯 IDENTIFYING RAABE'S TEST QUESTIONS
SIGNAL
Series has terms like [1·3·5...(2n-1)] / [2·4·6...(2n)] — products of odd over even numbers. Ratio test gives L=1, so Raabe's is needed.
SIGNAL
Question says "test for convergence" and the series ratio simplifies to something that → 1. The examiner WANTS you to apply Raabe's next.
TRICK
The formula n(1 - uₙ₊₁/uₙ) — first compute 1 - uₙ₊₁/uₙ, THEN multiply by n, THEN take limit. Don't try to shortcut!
📋 PYQ Questions — Raabe's Test
Q8
Test convergence: Σ [1·3·5·...·(2n-1)] / [2·4·6·...·(2n)]
🔥 Classic — Every Year 2019, 2020, 2021, 2022 7–14 Marks
▾
✏️ Step-by-Step
  • 1
    Write uₙ and uₙ₊₁
    uₙ = [1·3·5·...·(2n-1)] / [2·4·6·...·(2n)] uₙ₊₁ = [1·3·5·...·(2n-1)·(2n+1)] / [2·4·6·...·(2n)·(2n+2)]
  • 2
    Form ratio uₙ₊₁/uₙ (only NEW terms remain!)
    uₙ₊₁/uₙ = (2n+1)/(2n+2) KEY SHORTCUT: When products differ only in the last term, the ratio = (last term of numerator's next) / (last term of denominator's next)
  • 3
    Apply Ratio Test first
    L = lim(n→∞) (2n+1)/(2n+2) = lim(n→∞) (2 + 1/n)/(2 + 2/n) = 2/2 = 1 L = 1 → Ratio Test FAILS. Use Raabe's Test.
  • 4
    Apply Raabe's Test: compute n(1 - uₙ₊₁/uₙ)
    1 - uₙ₊₁/uₙ = 1 - (2n+1)/(2n+2) = (2n+2 - 2n-1)/(2n+2) = 1/(2n+2) n × [1 - uₙ₊₁/uₙ] = n/(2n+2) = n/(2(n+1))
  • 5
    Take the limit
    R = lim(n→∞) n/(2(n+1)) = lim(n→∞) 1/(2(1+1/n)) = 1/2 R = 1/2 < 1 → Series DIVERGES
✅ ANSWER

Series DIVERGES (Raabe's constant = 1/2 < 1)

💡 TRICK FOR PRODUCT SERIES

For series Σ [product of numerators] / [product of denominators], the ratio uₙ₊₁/uₙ = (next numerator term) / (next denominator term). You don't need to write out the full products — just identify what term gets added when you go from n to n+1.

√

Cauchy's Root Test

Use when terms are raised to the nth power — quick and clean
🧒 The nth Root Test

When each term uₙ looks like something raised to the power n (like (f(n))ⁿ), taking the nth root is the most natural operation. The Root Test asks: does (uₙ)^(1/n) approach something less than 1? If yes → converges. It's like asking "what's the average factor per term?"

√ Cauchy's Root Test
Compute: L = lim(n→∞) (uₙ)^(1/n) • If L < 1 → CONVERGES • If L > 1 → DIVERGES • If L = 1 → FAILS WHEN TO USE: uₙ has the form (f(n))ⁿ e.g. (n/(2n+1))ⁿ, (logn)ⁿ etc.
📋 PYQ Questions — Root Test
Q9
Test convergence of: Σ (n/(2n+1))ⁿ using Cauchy's Root Test
🟡 Medium Priority CT type question 7 Marks
▾
✏️ Step-by-Step
  • 1
    Identify uₙ: the entire term is raised to power n → Root Test
    uₙ = (n/(2n+1))ⁿ
  • 2
    Take nth root of uₙ
    (uₙ)^(1/n) = [(n/(2n+1))ⁿ]^(1/n) = n/(2n+1) This is the power of the Root Test — the n cancels perfectly!
  • 3
    Take limit
    L = lim(n→∞) n/(2n+1) = lim(n→∞) 1/(2+1/n) = 1/2 L = 1/2 < 1 → CONVERGES
✅ ANSWER

Series CONVERGES (L = 1/2 < 1 by Root Test)

📡

Power Series & Radius of Convergence

Every CSVTU ESE has at least one ROC question — very scoring!
🧒 The Radio Signal Analogy

A power series Σ aₙxⁿ is like a radio that works only within a certain distance R from the tower. Within R → signal is clear (convergent). Outside R → signal dies (divergent). At exactly distance R → could go either way (check separately). This "safe zone radius" R is called the Radius of Convergence.

📖 The Interval of Convergence

For power series Σ aₙ(x-a)ⁿ:
• Radius R = lim|aₙ/aₙ₊₁| (or 1/lim|aₙ|^(1/n))
• Series converges for |x-a| < R
• Series diverges for |x-a| > R
• At |x-a| = R (the endpoints): TEST SEPARATELY using other tests!
• Interval of convergence: (a−R, a+R) with endpoint checks

📡 Radius of Convergence Formulas
Method 1 — Ratio Method (most common in exams): R = lim(n→∞) |aₙ / aₙ₊₁| Method 2 — Root Method (Cauchy-Hadamard): R = 1 / lim(n→∞) |aₙ|^(1/n) Special cases: R = ∞ → series converges for ALL x R = 0 → series converges ONLY at center x = a
🎯 PATTERNS IN POWER SERIES QUESTIONS
FORM 1
Σ aₙ·xⁿ where aₙ has no x → Apply Ratio Test treating x as a constant. Solve for |x| < R.
FORM 2
Σ xⁿ/n² or Σ n·xⁿ → uₙ = aₙ·xⁿ, form uₙ₊₁/uₙ, keep |x| factor, solve.
FORM 3
"Find interval of convergence" → Find R first, then check both endpoints x=±R using other tests.
📋 PYQ Questions — Power Series
Q10
Find the radius of convergence and interval of convergence of: Σ (n·xⁿ)/(2ⁿ)
🔥 MUST DO ESE type — 2021, 2023 7 Marks
▾
✏️ Step-by-Step — Full Method
  • 1
    Identify aₙ (the coefficient without x)
    Series: Σ (n/2ⁿ) · xⁿ Here aₙ = n/2ⁿ and aₙ₊₁ = (n+1)/2^(n+1)
  • 2
    Compute R = lim|aₙ/aₙ₊₁|
    R = lim(n→∞) |aₙ/aₙ₊₁| = lim(n→∞) |(n/2ⁿ) / ((n+1)/2^(n+1))| = lim(n→∞) |n · 2^(n+1) / (2ⁿ · (n+1))| = lim(n→∞) |2n/(n+1)| = 2 · lim(n→∞) n/(n+1) = 2 × 1 = 2 Radius of Convergence R = 2
  • 3
    State convergence region
    Series converges for |x| < 2 → -2 < x < 2 Series diverges for |x| > 2
  • 4
    Check endpoints (x = 2 and x = -2)
    At x = 2: Σ n·2ⁿ/2ⁿ = Σ n → DIVERGES (→∞) At x = -2: Σ n·(-2)ⁿ/2ⁿ = Σ n·(-1)ⁿ → DIVERGES (terms don't→0) Both endpoints DIVERGE.
✅ FINAL ANSWER

Radius of Convergence R = 2
Interval of Convergence: −2 < x < 2 (open interval, both endpoints excluded)

💡 ENDPOINT CHECK — What to do every time

After finding R:
x = +R: Substitute x=R into original series, test with any standard test
x = -R: Substitute x=-R, check for convergence/divergence
Always state whether endpoints are included (closed bracket) or excluded (open bracket) in final answer!

Q11
Find radius of convergence: Σ (x-2)ⁿ / (n·3ⁿ) — series about a ≠ 0
🟡 High Priority Slightly harder PYQ variant 7 Marks
▾
🧒 Note: This is centered at a=2, not 0!

The series has (x-2)ⁿ instead of xⁿ. The method is identical — just treat (x-2) as a single unit. At the end, solve |x-2| < R to find the interval centered at 2.

✏️ Solution
  • 1
    Identify aₙ = 1/(n·3ⁿ)
    aₙ = 1/(n·3ⁿ), aₙ₊₁ = 1/((n+1)·3^(n+1))
  • 2
    R = lim|aₙ/aₙ₊₁|
    R = lim(n→∞) |[1/(n·3ⁿ)] / [1/((n+1)·3^(n+1))]| = lim(n→∞) (n+1)·3^(n+1) / (n·3ⁿ) = lim(n→∞) 3(n+1)/n = 3 R = 3
  • 3
    Interval: |x-2| < 3 → -1 < x < 5
    |x-2| < 3 -3 < x-2 < 3 -1 < x < 5 ← centered at 2, radius 3 Check x=5: Σ 1/n → DIVERGES (harmonic series) Check x=-1: Σ (-1)ⁿ/n → CONVERGES (alternating series test)
✅ ANSWER

R = 3, Interval of convergence: −1 ≤ x < 5 (left endpoint x=-1 included, right endpoint x=5 excluded)

n! in the series D'Alembert's Ratio Test — always Factorial cancels perfectly in ratio (n+1)!/n! = n+1

L = lim|u(n+1)/uₙ| L<1 → C, L>1 → D, L=1 → Fails
